SMILES simplified molecular-input line-entry system

Linear ASCII encoding: CCO is ethanol, c1ccccc1 is benzene. Fast to type, lossy on stereochemistry unless you use the isomeric variant.

InChI IUPAC int'l chemical identifier

Layered, deterministic string starting with InChI=1S/. One molecule, one canonical form. Built for unambiguous database identity.

InChIKey 27-char hash of InChI

Fixed-length SHA condensation: RYYVLZVUVIJVGH-UHFFFAOYSA-N. Designed for fast database keying and indexing.

CID pubchem compound identifier

Integer key into PubChem's catalogue of 100M+ compounds. Caffeine is 2519. Compact, citable, dereferenceable.
